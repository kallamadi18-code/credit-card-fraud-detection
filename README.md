# credit-card-fraud-detection
ML model to detect fraudulent credit card transactions

## 📋 Overview
Built a machine learning model to detect fraudulent credit card transactions using XGBoost and SMOTE. Achieved 87% fraud detection rate while handling extreme class imbalance (0.17% fraud cases).

## 🚀 Features
- Handles class imbalance using SMOTE
- XGBoost classifier for high accuracy
- 87% fraud detection rate
- 85 out of 98 frauds caught

## 🛠️ Technologies
- Python
- XGBoost
- SMOTE
- Scikit-learn
- Pandas

## 📊 Results
- Fraud Detection Rate: 87%
- Frauds Caught: 85/98
- False Alarms: 262
